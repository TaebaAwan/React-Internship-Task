## Prerequisites
To run this project locally, You should have:
-Node installed
-A code editor installed (preferably VScode)
-Git Installed

## Run project Locally
To run the project:
1) Clone it using:
```shell
git clone https://github.com/TaebaAwan/React-Internship-Task.git
```

2) install Node modules:
```shell
npm install
```

3) Run the project:
```shell
npm start
```
